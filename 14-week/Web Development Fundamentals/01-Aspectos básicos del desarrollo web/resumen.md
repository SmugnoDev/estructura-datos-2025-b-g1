# Principales puntos que recordar
1. Sea cual sea el tamaño de un ordenador, todos ellos tienen cuatro funciones básicas: entrada, proceso, salida y almacenamiento.
2. El código binario, también conocido como código de máquina, es un sistema que representa todos los datos como ceros o unos (falso o verdadero). Es el único lenguaje que un ordenador puede entender.
3. Los ordenadores necesitan hardware (los componentes físicos) y software (las instrucciones) para realizar las cuatro funciones.
4. El concepto de abstracción permite dar una instrucción informática sin comprender la enorme complejidad de la máquina ocultando los detalles innecesarios.
5. Un compilador convierte el código legible por las personas en código de máquina verificando la sintaxis y convirtiendo el código legible por las personas en código binario.
6. El proceso de desarrollo web consta de planificación y diseño, codificación, prueba, publicación y mantenimiento.
7. Los programas del lado del cliente se ejecutan en el dispositivo del usuario, mientras que los procesos del lado del servidor se ejecutan en los servidores del sitio web. 
8. Un desarrollador front-end crea todo lo que ve en una página web, como el texto y los botones.
9. Un desarrollador front-end utiliza principalmente HTML, CSS y JavaScript para crear sitios web.   
10. Un desarrollador back-end escribe el código en el que se basa la experiencia front-end, incluida la gestión de datos, y la reacción del servidor. 
11. Un desarrollador back-end utiliza JavaScript, Python y Java para desarrollar programas.
12. La World Wide Web está compuesta por todos los sitios web que usted utiliza. Internet es una red de ordenadores y servidores que proporcionan la infraestructura para acceder a la web.
13. Los desarrolladores web pueden utilizar servicios en la nube como ayuda para crear sitios web sólidos y escalables.
