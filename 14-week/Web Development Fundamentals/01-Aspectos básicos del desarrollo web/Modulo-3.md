# ¿ Que es el desarrollo web ?
### Descripción general del desarrollo web y del diseño web
    El diseñador se encargar de entender y prototipar lo que desea el cliente en su web.
    El desarrollador es el que se encarga de codificar el prototipo del diseñador.
### ¿ Por que es importante el desarrollo web ?
    El acelerado crecimiento de internet, ha hecho que millones de personas todos los dias utilicen paginas o aplicaciones web, no es de extrañar que se necesite desarrolladores para mantener y tambien crear soluciones web o simplemente entretenimiento, alli radica la importancia del desarrollo web.
### ¿ Que es un sitio web ?
    imagine que un sitio web es una colección de páginas web, todas ellas ubicadas en el mismo nombre de dominio.
- #### ¿Que es una pagina web?
        Una página web muestra información en Internet en forma de texto (copia), imágenes, vídeos y otros elementos.
        - El diseño.
        - El contenido.
        - El estilo y los colores.
        - La interactividad.
    
        Además de las páginas web, un sitio web también necesita:
        - Un nombre de dominio para poder localizarlo.
        - Un explorador para visualizar.
        - Un servidor para alojarlo.

### ¿ Como se desarrolla un  sitio web ?
    1. Planificacion y diseño.
    2. Codificacion.
    3. prueba y publicacion.
    
    Durante el desarrollo de una pagina web, el diseñador y el desarrollador generan entregables para llevar acabo el proceso, estos entregables suelen ser:

    1. Diseñador:
        - Planes de proyecto.
        - Historias de usuarios.
        - Maquetas y prototipos.
        - Mapa del sitio.
        - Diagramas de arquitectura.
        - Activos visuales.
    2. Desarrollador:
        - Codigo de servidor web.
        - Codigo del cliente.
        - Planes de prueba.
        - Imagenes del Cliente.
    
    Aunque no todo es crear desde 0, pues podemos llegar a trabajar en pagina con el fin de brindar soporte y mantenimiento.
### Proceso del lado del cliente y del lado del servidor
    Por lo general el cliente y el servidor estan separados, asi el frontend y el banckend componen un aplicativo web, el fontend se encarga de la funcionalidad visual (UI) y el backend se encarga de la logica del proceso y el almacenamiento.
