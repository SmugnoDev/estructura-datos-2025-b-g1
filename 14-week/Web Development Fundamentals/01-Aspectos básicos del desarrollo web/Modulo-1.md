# ¿Como funciona un ordenador?
### Funciones basicas de un ordenador
    1. Leer o recibir informacion (Input)
    2. Procesamiento de informacion (Process)
    3. Imprimir la informacion procesada (Output)
    4. Almacenar la informacion en memoria:
        * Volatil
        * No volatil
### Componentes del ordenador
    1. Hardware:
        - CPU
        - Memoria ram y ssd
        - Teclado, mouse, etc...
    2. Software:
        - De sistema (Windows, MacOS, Linux)
        - De aplicacion (word, google, zoom)
        - De programacion (vscode, eclipse, intellij)
### Conexión entre ordenadores y el desarrollo web
    Los navegadores web tales como google, firefox, etc..., son la conexión entre nuestro ordenador y el internet, siendo el desarrallo web aquellas intrucciones que le damos al navegador para interactuar con nuestra pagina web, pasando de pagina a aplicacion web ya que permitimos el acceso a los recursos del sistema.
